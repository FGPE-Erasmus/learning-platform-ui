## Requirements

* NodeJS
* Docker + Docker Compose

## Install & serve

```bash
$: npm install
$: npm serve
```

## Configuration

```bash
# Set your data in src/localSettings.js
$: cp src/localSettings.js.template src/localSettings.js
```
